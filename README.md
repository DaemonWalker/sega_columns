# Columns

一个基于 **Godot 4.6** 开发的本地 2D 益智游戏，灵感源自经典街机游戏《Columns》。

玩家控制由 3 颗宝石组成的垂直长条在 6×13 的网格中下落，通过水平移动、循环切换宝石顺序和加速下落，使 3 颗及以上同色宝石在纵向、横向或斜向相连并消除，触发连锁反应获得高分。

---

## 游戏特性

- **经典 Columns 玩法**：6 列 × 13 行网格，3 宝石垂直长条下落消除
- **四向匹配**：支持纵向、横向、左斜、右斜四个方向的同色消除
- **连锁反应**：消除后宝石下落填充空位，可触发连续消除，倍率翻倍（1× / 2× / 4× / 8×…）
- **等级系统**：每消除 35 颗宝石升 1 级，下落速度随等级提升
- **提示系统**：多种提示模式（全部可消除位置 / 随机一个 / 最高得分 / 最多消除）
- **排行榜**：本地持久化记录前 10 名的分数与消除数量
- **Demo 模式**：AI 自动演示游戏玩法
- **程序生成音效**：循环、锁定、消除等音效均由代码实时合成
- **键位/手柄自定义**：支持键盘和手柄输入，可在选项菜单中自由重映射

## 操作说明

| 动作 | 默认键位 |
|------|----------|
| 左移 | `←` 左方向键 |
| 右移 | `→` 右方向键 |
| 循环宝石 | `↑` 上方向键 / `Z` |
| 软降（加速下落） | `↓` 下方向键 |
| 暂停 | `Esc` |
| 确认/重启 | `空格` / `回车` |

> 所有键位均可在「选项 → 控制」中自定义，也支持手柄操作。

## 技术栈

- **引擎**：Godot 4.6（GDScript）
- **渲染**：GL Compatibility（2D 像素风）
- **分辨率**：640×480（支持窗口/全屏/无边框全屏切换）
- **字体**：Press Start 2P（像素风格）

## 运行项目

### 前置要求

- [Godot 4.6](https://godotengine.org/) 或更高版本

### 步骤

1. 克隆本仓库
2. 使用 Godot 编辑器打开项目根目录
3. 按 `F5` 运行主场景（`res://scenes/start_menu.tscn`）

### 导出

项目已配置 Windows Desktop 导出预设（`export_presets.cfg`），可通过 Godot 菜单「项目 → 导出」生成可执行文件。

## 项目结构

```
res://
├── scenes/              # 场景文件（.tscn）
│   ├── start_menu.tscn  # 开始菜单（主入口）
│   ├── single_player.tscn  # 游戏主场景
│   ├── options_menu.tscn   # 选项菜单

│   ├── music_player.tscn   # 背景音乐播放器
│   └── jewel.tscn          # 宝石预制体

│   ├── game_manager.gd     # 核心状态机、消除算法、分数系统
│   ├── active_column.gd    # 受控柱子逻辑
│   ├── hint_manager.gd     # 提示系统
│   ├── settings_manager.gd # 全局设置（Autoload）
│   └── leaderboard_manager.gd # 全局排行榜（Autoload）
├── assets/              # 游戏资源
│   ├── jewels/          # 7 种宝石纹理
│   ├── backgrounds/     # 背景图
│   ├── fonts/           # 像素字体
│   └── music/           # 背景音乐
└── tools/               # 辅助工具
    └── generate_jewels.py  # 使用 PIL 生成宝石纹理
```

## 注意事项

- 这是一个**纯本地单机游戏**，无网络通信、无用户账户系统。
- 设置与排行榜数据保存在平台相关的用户目录下（Windows 为 `%APPDATA%/Godot/app_userdata/Columns/`）。
- 所有 GDScript 脚本修改后直接在编辑器中运行即可验证，无需额外编译。
